Équité territoriale — Subventions d’État aux communes

Arrondissement de Lille, 2021–2025

Auteur·rice

Anton

Date de publication

24 avril 2026

Introduction

Ce rapport analyse la répartition des subventions d’État (DSIL, DETR, Fonds Vert, Recyclage Foncier…) entre les communes de l’arrondissement de Lille sur la période 2021–2025.

L’objectif est de mesurer l’équité territoriale : chaque commune a-t-elle reçu une part de l’enveloppe proportionnelle à son poids démographique ? Les écarts positifs signalent des communes sur-dotées ; les écarts négatifs, des communes sous-dotées.

2b. Financements actifs avec classification du dispositif

# Exclure les lignes annulées
financement_actif = financement[financement['is_annule'] != True].copy()

# Classification fund_type
def attribution_fund_type(type_fct):
    if pd.isna(type_fct):
        return 'other'
    t = str(type_fct).lower()
    if 'friche' in t: return 'RF'
    if 'vert'   in t: return 'FV'
    if 'fnadt'  in t: return 'FNADT'
    if 'dpv'    in t: return 'DPV'
    if 'dsil'   in t: return 'DSIL'
    if 'det'    in t: return 'DETR'
    return 'other'

financement_actif['fund_type'] = financement_actif['type_fct'].apply(attribution_fund_type)

# Jointure avec opérations pour récupérer sirene
fin_op = financement_actif.merge(
    operation[['id_operation', 'sirene', 'programmation']],
    on='id_operation', how='left'
)

4. Analyse d’équité

La méthode est simple : pour chaque commune, on calcule sa quote-part démographique (population / population totale de l’arrondissement), puis on en déduit le montant qu’elle aurait dû recevoir si la répartition avait été strictement proportionnelle.

L’écart = montant reçu − montant théorique.

Code 
analyse = sub_par_commune.merge(
    population[['COM', 'Commune', 'PMUN']],
    on='COM', how='left'
)
analyse['PMUN'] = pd.to_numeric(analyse['PMUN'], errors='coerce')

# Totaux intra-arrondissement (communes avec données population)
analyse_complete = analyse.dropna(subset=['PMUN']).copy()
total_sub = analyse_complete['montant_recu'].sum()
total_pop = analyse_complete['PMUN'].sum()

analyse_complete['poids_demo']        = analyse_complete['PMUN'] / total_pop
analyse_complete['montant_theorique'] = analyse_complete['poids_demo'] * total_sub
analyse_complete['ecart']             = analyse_complete['montant_recu'] - analyse_complete['montant_theorique']
analyse_complete['ecart_pct']         = (analyse_complete['ecart'] / analyse_complete['montant_theorique'] * 100).round(1)

analyse_complete = analyse_complete.sort_values('ecart', ascending=False)

# Affichage résumé
cols = ['commune', 'PMUN', 'montant_recu', 'montant_theorique', 'ecart', 'ecart_pct']
resume = analyse_complete[cols].copy()
for col in ['montant_recu', 'montant_theorique', 'ecart']:
    resume[col] = resume[col].apply(lambda x: f"{x/1e6:.2f} M€")
resume['PMUN'] = resume['PMUN'].apply(lambda x: f"{int(x):,}")
resume.columns = ['Commune', 'Population', 'Reçu', 'Quote-part théorique', 'Écart', 'Écart (%)']

print(f"Enveloppe totale analysée : {total_sub/1e6:.1f} M€  |  Population couverte : {int(total_pop):,} hab.")

6. Conclusion

Ce premier regard sur la répartition des subventions révèle des écarts significatifs entre communes. Ces disparités peuvent s’expliquer par plusieurs facteurs :

  • Capacité à déposer des dossiers : les grandes communes disposent de services techniques plus étoffés.
  • Nature des projets éligibles : certains dispositifs (DSIL) financent des projets d’envergure concentrés dans les pôles urbains.
  • Contexte politique et calendrier : les programmations annuelles influencent les attributions.

Une analyse complémentaire par dispositif (fund_type) et par année permettrait d’affiner ce diagnostic.
